🔧 Quick Usage Guide

Method 1: Convert Existing Notebook

python3 scripts/convert_notebook.py your-notebook.ipynb content/notebooks/your-content.md --title "Your Title"

Method 2: Create Content Manually

Create a new .md file in content/notebooks/ following this structure:

--- title: "Your Notebook" subtitle: "Learning Simulation" --- ## [MARKDOWN] Introduction content here ## [CODE] ```python print("Hello, world!") ``` **Output:** ``` Hello, world! ``` **Reflection:** ```json { "question": "What does this code do?", "type": "multiple-choice", "choices": [ { "text": "Prints a greeting", "correct": true, "feedback": "Correct! This prints 'Hello, world!' to the console." }, { "text": "Creates a file", "correct": false, "feedback": "Incorrect. This only prints text." } ] } ```

Method 3: View Your Content

templates/notebook-simulation-template.html?content=your-content.md

🎨 Customization Options

Styling: Edit styles/notebook-simulation.css to change colors, fonts, and layout.

Behavior: Edit js/notebook-simulation.js to add new cell types or question formats.

Content: All content is in markdown files in content/notebooks/ - no HTML knowledge needed!
