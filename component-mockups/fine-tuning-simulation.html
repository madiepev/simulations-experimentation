Step 1: Create Fine-Tuning Job

Navigate to Azure AI Foundry and initiate a new fine-tuning job for your model.

Azure AI Foundry > Model Catalog > Fine-tuning
Create fine-tuning job

What you'll see in Azure AI Foundry:

  • Navigate to the Model Catalog section
  • Click on "Fine-tuning" in the left navigation
  • Select "+ Create fine-tuning job" button
  • You'll see the job creation wizard interface

Azure AI Foundry Guide

About Azure AI Foundry
Azure AI Foundry is Microsoft's comprehensive platform for building, training, and deploying AI models. The fine-tuning feature allows you to customize foundation models for your specific use cases.
Navigation Tips
The left navigation panel is your main way to move between different Azure AI Foundry features. Key sections include Model Catalog, Fine-tuning, Deployments, and Monitoring.
Learning Objective
By following these steps, you'll become familiar with the Azure AI Foundry interface and understand how to navigate the fine-tuning workflow.
